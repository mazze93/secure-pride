## Key Principles

- **Privacy-First**: No telemetry, no tracking, no exceptions
- **SOGI Data Protection**: Highest standards for Sexual Orientation & Gender Identity data
- **Accessibility Built-In**: Designed for neurodivergent users from day one
- **Production-Ready Code**: SWE-bench Verified standard for all contributions
